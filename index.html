Welcome to MiniEPG Pages.

This automatic EPG generator is the easiest way to create a two day EPG for Telesystem BlobBox.

$ cd MiniEPG
$ Bash MiniEPG.sh
$ cp epg.v2.sqlite.zip /var/www

If you're using the Windows O.S., simply Install Cygwin on your system.

Automating

Put the previous lines in a script and add a line in your crontab file: 


10 6,18 * * * bash /home/MiniEPG/MiniEPG_cron.sh
